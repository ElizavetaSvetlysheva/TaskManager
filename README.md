Как запустить проект Важно! Правильная настройка запуска При открытии решения Visual Studio может назначить запускаемому проекту TaskManager.Core(библиотеку). Чтобы запустить приложение:
1. Открыть файл TaskManager.sln
2. В обозревателе решений правой кнопкой на TaskManager.WPF → Назначить запускаемым проектом
3. Нажать F5 или кнопку Запуск
